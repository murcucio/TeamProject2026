# Search Agent 역할 정의

Search Agent는 사용자가 입력한 주제에 맞는 논문 메타데이터를 수집하고, 다음 단계에서 바로 활용할 수 있도록 1차 품질 정제를 수행하는 역할을 담당한다.

## 주요 책임

- 사용자가 입력한 주제를 검색 키워드로 변환한다.
- arXiv와 Semantic Scholar에서 논문 메타데이터를 수집한다.
- 제목, 초록, 저자, 링크, 연도 등 필수 메타데이터를 정리한다.
- 주제 적합성, 초록 품질, 메타데이터 완전성, 중복 여부를 기준으로 1차 필터링을 수행한다.
- Reader Agent에 전달할 표준 JSON 구조로 저장 및 반환한다.

## 품질 기준

- 필수 메타데이터가 존재해야 한다.
- 초록이 비어 있지 않고 최소 길이 기준을 만족해야 한다.
- 검색 주제와 관련된 핵심 키워드가 제목 또는 초록에 포함되어야 한다.
- 컴퓨터공학 관련성이 낮은 논문은 제외한다.
- 중복 논문은 URL 또는 제목/연도 기준으로 제거한다.

## 출력 데이터

Search Agent는 정제된 결과를 `data/raw/search_result.json`에 저장하며, 이후 Reader Agent가 동일한 JSON 구조를 입력으로 사용한다.
